# Add Your Blog Images Here

## Quick Start

1. **Download your images** from [Google Drive](https://drive.google.com/drive/folders/19LIQam_Y8UCEY5P2iQXr3SzIA3jEGEh5?usp=sharing)

2. **Name them exactly like this:**
   - `summer-wedding-provencal-bastide.jpg` (featured)
   - `summer-wedding-provencal-bastide-1.jpg` (gallery photo 1)
   - `summer-wedding-provencal-bastide-2.jpg` (gallery photo 2)
   - etc.

3. **Drop them in this folder** (`public/images/blog/`)

4. **Refresh your browser** - images will appear automatically!

## Image Specs

- **Ratio**: 4:5 (portrait) - e.g., 1200 x 1500px
- **Format**: JPG or WebP
- **Size**: Under 500KB per image
- **Quality**: 85% is perfect
